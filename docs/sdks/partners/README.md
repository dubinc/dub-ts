# Partners
(*partners*)

## Overview

### Available Operations

* [create](#create) - Create a new partner
* [createLink](#createlink) - Create a link for a partner

## create

Create a new partner for a program. If partner exists, automatically enrolls them.

### Example Usage

```typescript
import { Dub } from "dub";

const dub = new Dub({
  token: "DUB_API_KEY",
});

async function run() {
  const result = await dub.partners.create();

  // Handle the result
  console.log(result);
}

run();
```
