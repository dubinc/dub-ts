# Continent

The 2-letter ISO 3166-1 code representing the continent associated with the location of the user.

## Example Usage

```typescript
import { Continent } from "dub/models/components";

let value: Continent = "AF";
```

## Values

```typescript
"AF" | "AN" | "AS" | "EU" | "NA" | "OC" | "SA"
```
